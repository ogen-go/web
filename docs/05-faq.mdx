## Answers

#### How to set `404 Not Found` handler?
Use `WithNotFound` option:
```go
h := myHandler{}
srv, err := api.NewServer(h, api.WithNotFound(func(w http.ResponseWriter, r *http.Request) {
		_, _ = io.WriteString(w, `{"error": "not found"}`)
}))
if err != nil {
	panic(err)
}
...
```
