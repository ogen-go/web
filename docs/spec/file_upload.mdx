OpenAPI 3.0 provides two ways to upload files:
- [File upload via request body](#file-upload-via-request-body)
- [File upload via `multipart` request](#file-upload-via-multipart-request)

## File upload via request body

File would be uploaded via request body.
The request body should be of type `string` and format `binary`.
